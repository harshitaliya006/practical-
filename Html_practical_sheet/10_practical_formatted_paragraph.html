HTML elements are thr building bolcks of HTML pages. With constructs, images and other objects such as interactive forms may be embedded into the rendered pag. HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraph, lists, links, quotes and ohter items. HTML elements are delineated by tags, written using angle brackets. Tags such as < img > and <input > directly introduce content into the page. Other tags such as < p> surround and provide information about document text and may include other tags as sub-elements. Browsers do not display the HTML tags but use them to interpret the content of the page.
